FAQ

Common questions

How does the AI investigation work?
When a detector finds an issue, RevBack automatically gathers context: the last 50 events for the affected user, all their entitlements, cross-platform identities, related issues, and similar issues from the past 24 hours. This context is sent to Claude (Anthropic's AI) which analyzes the data and returns a structured investigation with root cause, impact assessment, actionable recommendation, confidence score, and a reasoning chain explaining how it reached its conclusion. Results are cached for 24 hours.
Do I need an Anthropic API key?
No. AI features are included in Pro and Enterprise plans — we handle the AI infrastructure. The core detection engine (all 8 detectors, alerting, identity resolution) works fully without any AI dependency. AI investigation, incident clustering, and billing health insights are a premium layer that enhances the base product. Everything degrades gracefully if AI is unavailable.
Is my billing data sent to the AI?
AI investigations use subscription lifecycle events and metadata only — never credit card numbers, bank details, or PCI-scoped data. The data sent to Claude includes event types, timestamps, entitlement states, and user identifiers needed for root cause analysis. Anthropic does not train on your data. Enterprise customers can opt for on-prem deployment where no data leaves your infrastructure.
How long does setup take?
Most teams are fully set up in under 10 minutes. For Stripe, you connect with your API key and RevBack automatically backfills your last 30 days of subscription data. Apple, Google, and Recurly require pasting in API credentials, which typically takes about 5 minutes each. No code changes, SDKs, or deploys are needed. Your first AI investigation runs within minutes of connecting.
What if I only use Stripe?
RevBack delivers value even with a single billing provider. Many issues like webhook gaps, silent renewal failures, paid-without-access, and refund handling problems happen within a single platform. AI investigation and incident clustering work across all providers. Cross-platform detectors activate automatically when you add additional providers later.
How do you calculate revenue impact?
RevBack calculates revenue impact based on actual subscription data: the subscription price, billing interval, and number of affected users for each issue. AI investigations add churn risk assessment and aggregate impact when issues are clustered into incidents. All figures are based on real data, not estimates.
Can I self-host RevBack?
Yes. Enterprise plans include the option for on-premises or VPC deployment. RevBack is built on standard infrastructure (Node.js, PostgreSQL, Redis) and ships as a Docker container. In self-hosted mode, you provide your own Anthropic API key for AI features, or run without AI using only the rule-based detection engine.
